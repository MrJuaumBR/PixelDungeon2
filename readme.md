# Requirements
maxpygame >= 0.1.6.1
pygame
JPyDB >= 0.7.1

*MaxPyGame* - Engine
```shell
py -m pip install -i https://test.pypi.org/simple/ maxpygame
```

*JPyDB* - Database System
```shell
py -m pip install -i https://test.pypi.org/simple/ JPyDB
```

*PyGame* - SDL
```shell
py -m pip install pygame
```
